## Features

    </Card>
	<Card title="Built-in Editors" icon="pencil">
		Includes editors for creating game levels, animations, and managing game data.
	</Card>
	<Card title="Comprehensive Documentation" icon="open-book">
		Clear and complete documentation for easy learning and usage.
	</Card>
    <Card title="Rapid Prototyping" icon="rocket">
        Simple and easy-to-use engine for rapid prototyping and development of pixel art games.
    </Card>
